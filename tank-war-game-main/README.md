# 坦克大战 🎮

一个经典的坦克大战HTML5游戏。

## 游戏特点

- 玩家坦克 vs 敌方坦克
- 可破坏的墙壁障碍物
- 多关卡系统（击杀所有敌人进入下一关）
- 分数统计系统
- 流畅的操控体验

## 操作方式

- **方向键**: 移动坦克
- **空格键**: 射击
- **P键**: 暂停/继续
- **R键**: 重新开始

## 游戏规则

1. 消灭所有敌方坦克即可过关
2. 每击杀一个敌人获得100分
3. 被敌人子弹击中或与敌人碰撞会失去一条生命
4. 共有3条生命，用完则游戏结束

## 运行方式

直接用浏览器打开 `tank-battle.html` 文件即可开始游戏。

## 技术栈

- HTML5 Canvas
- 纯JavaScript实现
- 无需任何外部依赖
